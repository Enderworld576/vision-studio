How it works

  1. CollectPoint the camera at the part, draw a box around it, and save snapshots. Aim for 30+ from different angles.
  2. TrainOne click trains a model on your images. Takes a few minutes; progress shows live.
  3. TestSee the part detected live — its pixel location and orientation, relative to a default you set.

Import & export

Bring in images/models you already have, or export this dataset to use elsewhere. Imports and exports copy data, so this workspace stays self-contained.

Capture

Camera not streaming.
Capture a frame (or click a saved image to edit). Drag a box around each part; click a box to select it, then set its Direction or Delete it. For a full tray, use Grid fill.

Dataset

Train a model

All labeled parts are detected together — each becomes its own class, so they're learned apart automatically. Your negative-example images are included as background.
~15–20% of images are held out for validation (never trained on), so the accuracy below is honest. You can leave this tab; training keeps going.

Negative examples

Add images of look-alike objects that should NOT be detected as the selected part. They're used as hard negatives so the model learns to tell them apart. (No box needed — the whole image is the negative.)
